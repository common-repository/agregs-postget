== Description ==

    Este plugin é uma adaptação de um plugin chamado Post 2 Email.
    
	Basta instalar este plugin e enviar um e-mail para (e-mail aqui) para confirmarmos o seu cadastro e todo novo post  no seu blog fará parte, automaticamente, do agregador automático adulto agregs.com.
    
	Você deve ter recebido uma cópia da licença junto com o wordpress.

== Installation ==

1. Baixe o plugin pelo repositório de plugins do wordpress ou extraia o .rar com o plugin para `/wp-content/plugins/`.
2. Ative o plugin no seu wordpress.
3. Nanegue para Configurações->Leitura e preenchecha o campo agregsPostGet com o seu e-mail.
4. Envie um e-mail para contato@aggregs.com com o e-mail acima e o endereço do blog/site a ser cadastrado.
5. Pronto, a partir de agora os novos posts serão agregados automaticamente no Agregs.com.

== Frequently Asked Questions ==

Q:Onde cadastro o e-mail?
R:O campo para cadastramento do e-mail está em Configurações->Leitura.

Q:É só colocar o e-mail lá e estarei cadastrado?
R:Não, por uma questão de segurança os e-mails devem ser aprovados manualmente; envie o e-mail que foi cadastrado, juntamente com o endereço do site/blog em questão para contato@agregs.com.
